# Publicación en Vercel

1. Sustituye en GitHub los archivos actuales por esta versión.
2. Haz Commit changes.
3. Vercel detectará el cambio y publicará automáticamente.
4. Si no se actualiza solo: Vercel → Deployments → Redeploy.

## Comprobación

Cuando termine el despliegue, abre:

```txt
https://www.inmobiliariaelpilar.com/api/listings
```

Si todo está bien, verás un JSON con `count` y `listings`.

Después entra en la web y baja a **Propiedades**: deben aparecer los inmuebles dentro de la propia web.
